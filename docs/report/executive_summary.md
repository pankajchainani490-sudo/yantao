# 项目总结摘要

## 项目名称

基于机器学习的恶意流量识别系统

## 项目目标

本项目面向正常流量、ARP 攻击、DDoS 和木马流量的识别需求，构建了一个集数据规范、特征工程、机器学习分类、告警预警、黑名单管理、前后端展示以及 Ubuntu 部署准备于一体的完整系统。

## 已完成内容

### 1. 数据与标签规范

- 设计了统一的数据目录结构
- 设计了四类流量标签映射
- 提供了样本来源清单与回放清单

### 2. 特征工程

- 完成基础流量特征与衍生特征设计
- 实现样本数据到处理后训练数据的流水线

### 3. 模型训练

- 实现决策树与随机森林训练
- 导出模型文件、混淆矩阵与特征重要性结果

### 4. 后端系统

- 实现推理接口、仪表盘接口、告警接口、黑名单接口、回放接口、指标接口和设置接口
- 使用 SQLite 保存预测记录、告警、黑名单与回放状态

### 5. 前端系统

- 实现 Dashboard、Detection、Replay、Blacklist、Metrics、Settings 六个页面
- 前端已接入真实后端接口

### 6. 测试与验证

- 基于 pytest 建立分阶段自动化验证体系
- 已覆盖结构、数据、特征、训练、接口和部署准备验证

### 7. 部署准备

- 补齐 Ubuntu 部署说明
- 补齐 nginx、systemd、docker-compose 模板

## 当前项目特点

- 从数据到展示形成完整闭环
- 同时兼顾可解释性、可展示性和可部署性
- 适合作为课程设计、毕设原型或安全演示项目

## 后续扩展方向

- 接入更复杂和更真实的数据集
- 加入深度学习模型进行对比
- 增强实时抓包和在线识别能力
- 扩展更丰富的大屏图表和安全运营联动
